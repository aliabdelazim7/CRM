Google Sheets Integration

Leave empty to run in offline Mock Mode using local storage.

Setup Guide

Connect Google Sheets in 3 steps:

  1. Open a Google Sheet, and select Extensions > Apps Script.
  2. Clear existing script, copy code from google-apps-script/code.js, and paste it.
  3. Click Deploy > New Deployment. Choose Web App, change Access to Anyone, and deploy. Copy the Web App URL and paste it left.
Note: When you connect the API URL, the system will download sheet data and run. You can toggle off the URL at any time to run independently.
